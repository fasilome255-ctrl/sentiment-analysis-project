# Sentiment Analysis Project

## Overview
This project performs sentiment analysis on text data using machine learning models.

## Models Used
- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)

## Techniques
- CountVectorizer
- TF-IDF
- ROC-AUC evaluation
- Confusion Matrix

## Dataset
Twitter Sentiment Dataset (public dataset)

## Results
Logistic Regression performed best with balanced accuracy and interpretability.
